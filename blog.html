Short Detail about API

var

Function-scoped. Can be redeclared and updated. Hoisted to the top of its scope.

let

Block-scoped. Cannot be redeclared within the same block but can be updated. Not hoisted.

const

Block-scoped. Cannot be redeclared or updated. The value it holds cannot be reassigned, but if it's an object, the object's properties can be changed.

null 

let user = null;

Represents the intentional absence of any object value. Use it when you want to intentionally clear a variable, indicating that it does not point to any object.

undefined 

let user;
  console.log(user); // undefined

Indicates a variable has been declared but has not yet been assigned a value. It is also the default return value for functions that do not explicitly return something.

REST API

A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for building and interacting with web services. It uses standard HTTP methods (GET, POST, PUT, DELETE) to perform CRUD (Create, Read, Update, Delete) operations on resources, which are typically represented in JSON or XML format. REST APIs are stateless, meaning each request from a client to server must contain all the information needed to understand and process the request.
